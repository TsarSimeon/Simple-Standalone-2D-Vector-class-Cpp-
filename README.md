Simple-Standalone-2D-Vector-class-C-

A lightweight standalone 2D vector class written in C++ for use in small projects/game dev

Features:
-> Vector addition and subtraction
-> Scalar multiplication and division
-> Dot product
-> Length calculation
-> Normalization
-> Compound operators (+=, -=, *=, /=)

Usage:
include in your project
#include "Vec2.hpp"

How to build:
No need to build it's header-only file

Linking:
Automatic as long as the compiler knows where the header is (e.g. via include path)
